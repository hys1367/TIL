##숫자형(Number)
숫자로 이루어진 자료형 정수나 실수 등을 다룰 수 있다. 숫자 끼리의 연산이 가능하다.
##문자열(string)
문자나 문자들을 늘어놓은 것
큰 따옴표(")와 작은 따옴표(')로 구분

'Hello!'
'3.14'
"3.14"
##리스트(List)
여러 자료를 보관하는 자료형.
다른 종류의 자료를 함꼐 담을 수 있다.
자료 안에 순서가 있다.(문자열이1번째 숫자가 2번째)

[] #빈리스트
['a', 'b']
['a', 2]
