##변수
자료를 '그릇'에 담아서 보관, 사용하면 편리하다. 이를 변수(variable)라 한다.

####변수이름=자료
num=10 #숫자
name="yoonseong" #문자열
grade=['a+', 'b+', 'a0'] #리스트
###사용 가능한 변수 이름
숫자,알파벳,한글,언더바(_)등
###사용 불가한 변수이름
1. 변수이름이 숫자로 시작하며 안된다.
2. 숫자로만 구성된 변수 이름 금지
3. 파이썬 문법에서 사용되는 예약어
(이미 사용되는 단어 : for, while,if)사용 금지
4. 공백문자()와 연산자(+,-,% 등)사용금지
